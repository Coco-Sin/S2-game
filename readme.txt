Bienvenue dans le jeu “Escape Game”, un jeu réalisé dans le cadre d’un projet scolaire en programmation procédurale.

## ----- Principe ----- ##

    ▪	Le but du jeu est de sortir de la pièce
    ▪	Pour se faire, vous devez résoudre chaque énigme et réunir chaque indices récoltés dans une   ultime question final
    ▪	Vous êtes seul pour résoudre chaque énigme
    ▪	Vous avez 45 minutes pour sortir de la pièce (ce qui est largement suffisant). Une fois le temps écoulé, vous perdez la partie.
    ▪	Pour trouver des énigmes et des indices, vous devez explorer le menu des objets, représentés par des X sur la carte principale
    ▪	Vous avait biensûr le droit à une et une seule aide par énigme, en cas de soucis (ne vous inquiétez pas, tout se passera bien ;) )

## ----- Touches ----- ## 

    ▪	z   :  permet de se déplacer vers le haut
    ▪	q   :  permet de se déplacer vers la gauche
    ▪	s   :  permet de se déplacer vers le bas
    ▪	d   :  permet de se déplacer vers la droite
    ▪	h   :  permet d’accéder à l’aide associée à l’énigme en cours
    ▪	m   :  permet de retourner à la carte principale (ou 12 pour la porte de sortie0
    ▪	i   :  permet d’accéder aux informations concernant le temps restant et un récapitulatif du Readme (règles et touches)


SURTOUT N'OUBLIEZ PAS D'ENTRER VOTRE RÉPONSE UNE FOIS LA LETTRE OU LE NUMERO SÉLECTIONNER!
